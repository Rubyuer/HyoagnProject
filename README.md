# 视立体育

  应用介绍:"视立体育"app是一款线上购买网球商品、预约网球场馆的应用。

  应用功能:
  
    1.商品：网球/球拍、运动护具；
       用户下单成功后需到"中海兰庭网球俱乐部"取货。
       
    2.场馆：室外网球场、室内网球场；
       用户预约后需在预约的时间内到达场馆进行使用。

    3.订单：商品订单(全部订单/待取货/已完成)、我的预约(全部预约/预约中/已消费)。
    
    4.我的：商品收藏、优惠券、场馆收藏、我的消息、我的设置和我的个人信息。
